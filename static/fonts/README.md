# Inter

These `.woff2` files are the [Inter](https://rsms.me/inter/) typeface (latin subset,
weights 400–800), bundled so the app needs no external font CDN.

Inter is licensed under the **SIL Open Font License 1.1** and may be freely bundled and
redistributed. License: https://github.com/rsms/inter/blob/master/LICENSE.txt

Font files sourced from the [@fontsource/inter](https://www.npmjs.com/package/@fontsource/inter)
package.
